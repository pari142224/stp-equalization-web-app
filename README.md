# STP + Equalization Tank Design Tool

Interactive Web-Based STP Design Software based on CPHEEO guidelines.

## Features
- Peak Flow Based Design
- 24 Hour Equalization Tank Design
- Ripple Diagram Plot
- Blower Capacity Calculation
- Tank Dimension Suggestion

## Run Locally

pip install -r requirements.txt
streamlit run app.py
